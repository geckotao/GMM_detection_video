# 视频画面变化检测工具

本工具基于高斯混合模型（GMM）算法，自动检测视频中画面变化区域。

当变化比例超过设定阈值时，自动保存当前帧截图，适用于安防监控、视频审查、事件回溯等场景。

支持批量处理、关注区域（ROI）限定、处理倍速调节、截图间隔控制等功能。

- 核心算法：cv2.createBackgroundSubtractorMOG2（GMM）

- 融合策略：GMM 前景 ∩ 帧间差分 → 连通区域过滤

- 分辨率：自动缩放至高度 ≤480P加速处理（可修改代码调整）

【功能概览】

- 批量添加/移除/预览视频文件（支持 MP4、AVI、MOV、MKV、FLV）

- 手绘多边形关注区域（ROI），仅对该区域进行变化检测

- 实时预览检测结果（红色高亮变化区域）

- 可调参数：

  • 画面变化阈值（0.1% ~ 50%）

  • 截图最小间隔（0.1 ~ 10 秒）

  • 处理倍速（1x ~ 64x，通过丢帧加速）

  • GMM 敏感度（varThreshold）

  • 帧间差分阈值（用于抑制“鬼影”误报）

- 变化率大于设定值自动保存截图至指定文件夹

- 详细日志记录（位于“检测日志”文件夹）
